## ⚡ Ключевые особенности

* **🧬 Quantum Bypass Logic:** Изолированные на уровне ядра блоки перехвата `try-catch`, полностью предотвращающие критические вылеты (краши) игры при обработке нестандартных кодировок (например, кириллических строк CP1251 в игровом чате).
* **🎯 Multi-Type Interceptor:** Перехват, валидация и подмена данных «на лету» на уровне встроенного движка Lua (функции `lua_pushlstring`, `lua_pushstring`, `lua_pushnumber`, `lua_pushinteger`).
* **📱 Smart Mobile Sync:** Интеллектуальный модуль обнаружения Android-устройств через ADB. Самостоятельно разворачивает необходимую файловую структуру и доставляет компоненты в один клик.
* **🔍 X-Ray Logging:** Прозрачная низкоуровневая трансляция сигнатур, вызовов функций и внутренних состояний памяти в изолированный файл дампа `lua_xray_dump.txt`.

## 🛠️ Архитектура комплекса

Система разделена на два независимых слоя автоматизации:
1. **`launcher.py` (Python)** — диспетчер управления. Отвечает за низкоуровневую связь с устройством через ADB, инициализацию конфигурации, выбор профилей разрядности лаунчера (x32/x64) и стабильное удержание Frida-сессии.
2. **`hook.js` (JavaScript)** — инъекционный скрипт. Внедряется в рантайм-процесс библиотеки `libmonetloader.so`, изолирует целевые участки кода проверки и генерирует валидные структуры ответов (активация VIP-статусов).

## 🚀 Быстрый старт

### 1. Подготовка окружения
Убедитесь, что в системе установлен Python версии 3.10 или выше, затем установите зависимости через терминал:
```bash
pip install frida-tools

```

### 2. Первичный запуск

Включите режим **«Отладка по USB»** на вашем Android-устройстве, подключите его к ПК и запустите управляющий модуль:

```bash
python start.py

```

*Интерактивный мастер конфигурации автоматически определит серийный номер смартфона и предложит выбрать игровой профиль.*
серийный номер смартфона - сохраняется только для сохранения настроек лаунчера не более

### 3. Сброс настроек

Для смены целевого устройства или переключения профилей лаунчера используйте флаг сброса:

```bash
python launcher.py --reset

```

## 📂 Структура репозитория

```text
├── adb/                    # Локальные бинарные файлы Android Debug Bridge
├── monetloader/            # Компоненты автоматической доставки на устройство
│   ├── libfrida-gadget.so  # Гаджет Frida под целевую архитектуру (ARM/ARM64)
│   ├── frida_start.lua     # Первичный загрузчик среды окружения
│   └── Arizona Helper.lua  # Анализируемый целевой Lua-скрипт
├── config.json             # Автоматически генерируемый профиль конфигурации сессии
├── hook.js                 # Основной JS-скрипт инжекции и перехвата вызовов
└── start.py                # Python-лаунчер координации и мониторинга

```
