Проект разделен на две логические части:

## 🚀 Быстрый старт

### 1. Подготовка окружения
Убедитесь, что у вас установлены Python 3.10+ и зависимости:
```bash
pip install frida-tools

```

### 2. Первичный запуск

Подключите Android-устройство по USB (включив "Отладку по USB" в настройках разработчика) и запустите лаунчер:

```bash
python launcher.py

```

Мастер первоначальной настройки автоматически зафиксирует серийный номер устройства и предложит выбрать профиль вашей игры.

### 3. Сброс конфигурации

Если вам нужно сменить устройство или переключить профиль лаунчера, запустите скрипт с флагом сброса:

```bash
python launcher.py --reset

```
